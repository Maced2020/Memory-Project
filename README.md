# Videogame Memory Game

## Project Purpose:

This game was built for the final Udacity Course project. The purpose of the project is to demonstrate JavaScript, HTML, and CSS

## How to Load the game

- unzip the Folder and open index.html

### How to Play the Game

The game board consists of twelve to twenty-four cards arranged randomly in a grid. The deck is made up of six to twelve pairs of cards, each with different Pictures to match up.

At start player chooses a level:

- "easy" needs to match 4 pairs of cards
- "medium" needs to match 6 pairs of cards
- "hard" needs to match 12 pairs of cards

On each turn:

- The player flips one card over to reveal its underlying Console
- The player then turns over a second card, trying to find console that matches
- If the Game cards match, both cards stay flipped over
- If the Game cards don’t match, both cards are returned to their hidden state
- once all cards have been matched The game ends.


### Special Features

- Player choose one of three levels "easy", "medium" and "hard"
- The game contains a timer to keep track how long completing the game takes 
- Each player starts with a three star rating. The more moves the player makes the Lower the rating will be 
- At games end a pop-up shows with the completion time, star rating and a “Play again” game button
